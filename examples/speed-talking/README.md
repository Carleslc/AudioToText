## Example: Speed Talking

This example transcribes an audio file using Whisper to its source language (English) in TXT, VTT, SRT, TSV and JSON formats.

## Input

**task**: `Transcribe`

**audio_file**: [`micro-machines.wav`](micro-machines.wav)

**use_model**: `medium`

**language:** `English`
